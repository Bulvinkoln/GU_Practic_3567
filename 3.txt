Работа с Гит

**git init**
создает новый репозиторий Git

**git add 1.txt**
добавляет изменение в файле 1.txt(или весь его, если до этого не был включен) в рабочий каталог в промежуточную область Git.

**git add -a** 
**git add .**
эквивалентны
новые (или неотслеживаемые), удаленные и измененные файлы будут добавлены в промежуточную область Git.

**git commit -m "Текст коммита"**
записывает индексированных изменений в репозиторий Git

**git commit -a -m "Текст коммита"**
добавляет изменения и  записывает их в репозиторий Git

**git diff**
показывает разницу между рабочей копией и индексом

**git log**
перечисляет коммиты, сделанные в репозитории в обратном к хронологическому порядке

**git branch имя ветки**
создание ветки, без переключения на нее

**git checkout имя ветки**
переключение на ветку

**git checkout -b имя ветки**
создание ветки, с переключением на нее

**git clone путь к репозиторию путь к папке**
клонирование репозиторя находящегося по указанному пути на папку локального ПК 

**git pull исходная ветка наша ветка**
извлечение и загрузка содержимого из удаленного репозитория и немедленное обновление локального репозитория этим содержимым 

**git push наша ветка исходная ветка**
выгрузка содержимого локального репозитория в удаленный репозиторий
